# Boards i18n Completion Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire `useTranslations` into the remaining 7 boards components so the entire Boards feature switches between PT and EN.

**Architecture:** Follow the established pattern — `import { useTranslations } from 'next-intl'` at the top of each file, call `const t = useTranslations('boards.<namespace>')` inside the component, replace every hardcoded PT string with `t('key')`. For new namespaces (KanbanHeader, DeleteBoardModal, AIProcessingModal) add the EN keys to `messages/en.json` and the PT keys to `messages/pt.json` first, then wire. Keys already exist for KanbanBoard, DealCard, BoardCreationWizard and DealDetailModal — just wire them.

**Tech Stack:** Next.js App Router, React 19, TypeScript, `next-intl`, Vitest, React Testing Library.
